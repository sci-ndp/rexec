Authentication

All write operations and protected endpoints require a Bearer token in the Authorization HTTP header.

Bearer Token Scheme

Authorization: Bearer <your-token>

Tokens are validated against the configured AUTH_API_URL. The auth API must accept a GET request with the same Authorization header and return a JSON body with at minimum: 

// Expected AUTH_API_URL response shape
{
  "username": "alice",
  "groups": ["/ndp_ep/ep-<id>", "researchers"]
}
ℹ️
Test IDP https://idp-test.nationaldataplatform.org/temp/information can be used during development. GET endpoints are generally public (no token required).

Rexec Server Deployment API

FastAPI service that provisions and destroys per-user rexec-server pods in Kubernetes. By default served at your configured Ingress host under path /rexec.

Interactive Swagger UI: https://<your-host>/rexec/docs

GET /rexec/health Service health check ▼

Returns 200 OK with a simple status message. No authentication required.

Response

HTTP/1.1 200 OK
Content-Type: application/json

{"status": "ok"}
GET /rexec/servers List active server pods ▼

Returns a list of currently provisioned rexec-server pods across all user namespaces. Public endpoint — no auth required.

Response

HTTP/1.1 200 OK

[
  {
    "namespace": "rexec-server-alice",
    "pod_name":   "rexec-server-alice-7d4f9b",
    "status":    "Running",
    "created_at":"2025-03-14T03:00:00Z"
  }
]
POST /rexec/spawn 🔑 Auth required Provision a server pod ▼

Creates a new rexec-server pod in a dedicated namespace (rexec-server-<username>). If a pod already exists for this user, returns the existing pod info.

🔑
Requires Authorization: Bearer <token>. If group-based ACL is enabled, the token's groups must include a configured GROUP_NAMES entry.

Request Headers

HeaderRequiredValue
AuthorizationRequiredBearer <token>

Response

HTTP/1.1 200 OK

{
  "namespace":  "rexec-server-alice",
  "pod_name":   "rexec-server-alice-7d4f9b",
  "status":    "Running",
  "broker_url": "tcp://<node-ip>:30001"
}
DELETE /rexec/server 🔑 Auth required Tear down server pod ▼

Tears down the authenticated user's rexec-server pod and deletes its namespace. Idempotent — returns 200 even if no pod existed.

Request Headers

HeaderRequiredValue
AuthorizationRequiredBearer <token>

Response

HTTP/1.1 200 OK

{"message": "Server for user 'alice' deleted."}

NDP Endpoint API

FastAPI service providing dataset discovery, authentication integration, and Rexec spawn API address to client libraries. Served at /api by default.

Interactive Swagger UI: https://<your-host>/api/docs

GET /api/health Service health check ▼

Health and readiness check. Returns 200 when the API is ready. No auth required.

HTTP/1.1 200 OK

{"status": "ok", "endpoint": "My NDP Endpoint"}
GET /api/rexec Get Rexec connection info ▼

Returns the broker address and spawn API URL for the rexec client library to connect and provision a server. This is what the ndp_ep client calls internally when you use ep.rexec.run().

Response (Rexec enabled)

HTTP/1.1 200 OK

{
  "rexec_enabled":       true,
  "broker_url":           "tcp://<node-ip>:30001",
  "spawn_api_url":        "https://your-host/rexec"
}

Response (Rexec disabled)

HTTP/1.1 200 OK

{"rexec_enabled": false}
GET /api/datasets Search dataset catalog ▼

Search the local and global dataset catalogs. See the ep-api documentation for full query parameters.

Query Parameters

ParameterDescription
qoptionalFree-text search query
limitoptionalMax results (default: 10)
offsetoptionalPagination offset
 
GET /api/datasets?q=climate&limit=5

HTTP/1.1 200 OK

{
  "count":   42,
  "results": [...]
}

HTTP Error Codes

CodeMeaningCommon Cause
200OKRequest succeeded.
400Bad RequestMalformed request body or missing required field.
401UnauthorizedMissing or expired Bearer token.
403ForbiddenToken is valid but the user's groups are not in the ACL allow-list.
404Not FoundResource (e.g., server pod) does not exist.
409ConflictA server pod already exists for this user.
500Internal Server ErrorKubernetes API error or misconfiguration. Check pod logs.
503Service UnavailableAuth API or Kubernetes cluster not reachable.
